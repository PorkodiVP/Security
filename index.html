INFRASTRUCTURE AND SECURITY PILLAR

Security Pillar encompasses security due to code vulnerabilities in our software application, vulnerabilities in OSS used in our applications, and infrastructure security.

  • Code vulnerabilities are detected and tracked via world class code quality and security analysis tool like Sonarqube. It is tightly integrated into the development process to prevent security vulnerabilities to be introduced into our applications.

  • OSS vulnerabilities are constantly monitored so that they are quickly identified and fixed to ensure customers get fixes to these vulnerabilities as soon as possible.

  • Infrastructure security ensures that all systems and services used by the R&D organization are properly secured.
